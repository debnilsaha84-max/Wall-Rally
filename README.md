
# Wall Rally

Wall Rally is a simple arcade game built in Python for Stanford Code in Place 2026.

## How to Play

* Use the **LEFT** and **RIGHT** arrow keys to move the racket.
* Keep the ball from touching the ground.
* Each successful hit increases your score.
* The ball speeds up as your score increases.

## Features

* Start screen with instructions
* Score tracking
* Increasing difficulty
* Game Over screen
* Final score display

## Technologies Used

* Python
* Stanford Code in Place Graphics Library
